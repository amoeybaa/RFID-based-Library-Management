General Library Rules


  1. Membership Requirements: Users must have a valid library membership to access online resources. Membership is limited to specific user groups (students, faculty).
  2. Access Hours: Library is accessible during college hours.
  3. Respect Copyright Laws: Users must adhere to copyright laws and licensing agreements when using digital materials.
  4. Privacy and Data Security: Ensure user data privacy and security.
  5. Borrowing and Return Policies:
    • Students: Can borrow upto 2 books for a maximum of 2 weeks.
    • Teachers: Can borrow upto 3 books for an extended amount of time.
  6. Penalties for Late Returns: Explain consequences for not returning digital items on time, such as suspension of borrowing privileges.
  7. Policy Enforcement: Explain how the library enforces these rules and any consequences for violating them.
